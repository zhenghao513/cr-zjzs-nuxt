# Nuxt - 浙江省成人高考报名

查看 [Nuxt 3 文档](https://nuxt.com/docs/getting-started/introduction) 了解更多信息。

## 安装

- 确保安装了依赖项:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## 开发服务器

在 `http://localhost:3000` 上启动开发服务器

```bash
npm run dev
```

## 生产

构建用于生产的应用程序：

```bash
npm run build
```

- 本地预览生产构建：

```bash
npm run preview
```

查看 [部署文档](https://nuxt.com/docs/getting-started/deployment) 获取更多信息。
